# Student Performance Analysis

## Overview
This project analyzes student performance using Python, Pandas, and Matplotlib.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Analysis Performed
- Data exploration and cleaning
- Parental education analysis
- Test preparation analysis
- Correlation analysis
- Gender-wise comparison
- At-risk student identification
- Data visualizations

## Key Findings
- Test preparation improves student performance.
- Reading and writing scores are strongly correlated.
- Male students perform better in math.
- Female students perform better in reading and writing.
- 188 students were identified as at risk.
